# Student Grade Tracker

## 📌 Project Description
Student Grade Tracker is a Java console application developed as part of the CodeAlpha Java Programming Internship. It allows users to manage student records, calculate grades, and generate a summary report.

## 🚀 Features
- Add student details
- Store marks using ArrayList
- Display all student records
- Calculate average marks
- Find highest and lowest marks
- Console-based user interface

## 🛠 Technologies Used
- Java
- Object-Oriented Programming (OOP)
- ArrayList
- Scanner

## ▶️ How to Run
1. Open the project in VS Code, Eclipse, or IntelliJ IDEA.
2. Compile the Java file.
3. Run `StudentGradeTracker.java`.

## 📷 Sample Output
The application displays student records along with average, highest, and lowest marks.
